 Pharmacy Dashboard (Sales, Profit & Inventory Analysis) – Power BI Project
This project focuses on analyzing pharmacy sales, profitability, inventory levels, and medication expiry status using Power BI. The dashboard is designed as a single, interactive view that supports data-driven decision-making for pharmacy management.
![Image](https://github.com/user-attachments/assets/17bd6e91-e42b-41d6-8479-9aaaa40e4c3f)

🔹 Project Objectives
The dashboard addresses three main analytical goals:
1.	Financial Performance Analysis
o	Analyze total sales, total profit, average profit margin, and quantities sold.
o	Track year-over-year (YoY) performance to evaluate business growth.
o	Identify the top profitable medicines to support pricing and sales strategies.
2.	Product & Inventory Analysis
o	Monitor product quantities and inventory distribution.
o	Highlight inventory status across valid, near-expiry, and expired medicines.
o	Support efficient stock management and demand planning.
3.	Medication Expiry Tracking
o	Track medicines based on expiry dates and days remaining to expiration.
o	Identify near-expired and expired products to reduce waste and financial loss.
o	Provide a detailed table for operational follow-up.
🔹 Key Features
•	Interactive KPIs for Sales, Profit, Quantity, and Profit Margin
•	Dynamic slicers (Year, Region, Brand, Customer Type) for flexible analysis
•	Visual breakdown of profit by customer type and region
•	Top 10 medicines by profit and expiry status analysis
•	Clean and user-friendly dashboard layout following Power BI best practices
🔹 Tools & Skills Used
•	Power BI
•	Data modeling and relationships
•	DAX measures for KPIs and YoY calculations
•	Data visualization and dashboard design
